== Description ==

WordPress plugin.
Adds an FAQ section with jQuery UI accordion, supports native options, categories, duplication, and reordering.

== How to use ==

Add shortcode to your page or post:
[faq_accordion animate="400" collapsible="true" active="false" category="about"]

It uses normal jQuery UI accordion options and supports optional categories.
If no category slug is provided, it will display all FAQs.
